# 数学公式可视化文档

## 概述

星辰项目实现了完整的数学公式可视化系统，支持多种混沌吸引子的实时渲染和交互。粒子根据数学方程在 3D 空间中运动，创造出优美的视觉效果。

## 支持的数学模型

### 1. Lorenz 吸引子

**历史背景**
- 发现者：Edward Lorenz（1963）
- 背景：研究大气对流时发现的混沌系统
- 意义：第一个被发现的混沌系统

**数学方程**
```
dx/dt = σ(y - x)
dy/dt = x(ρ - z) - y
dz/dt = xy - βz
```

**参数说明**
- σ (sigma)：普朗特数，控制对流强度，典型值 10
- ρ (rho)：瑞利数，控制温度差，典型值 28
- β (beta)：几何因子，控制纵横比，典型值 8/3

**视觉效果**
- 经典的蝴蝶形状
- 双翼结构
- 粒子围绕两个中心点螺旋运动

**应用场景**
- 气象学研究
- 流体力学模拟
- 混沌理论教学

### 2. Thomas 吸引子

**历史背景**
- 发现者：René Thomas
- 背景：研究循环对称混沌系统
- 意义：展示循环对称性的混沌行为

**数学方程**
```
dx/dt = sin(y) - bx
dy/dt = sin(z) - by
dz/dt = sin(x) - bz
```

**参数说明**
- b：阻尼系数，典型值 0.208186

**视觉效果**
- 三个对称的螺旋臂
- 循环对称结构
- 平滑的周期性运动

**应用场景**
- 化学反应动力学
- 生物学模拟
- 对称性研究

### 3. Clifford 吸引子

**历史背景**
- 发现者：Clifford Pickover
- 背景：研究 2D 平面上的混沌图案
- 意义：展示简单方程产生的复杂图案

**数学方程**
```
dx/dt = sin(a*y) + c*cos(a*x)
dy/dt = sin(b*x) + d*cos(b*y)
dz/dt = sin(x) + cos(y) + z*0.1
```

**参数说明**
- a、b：频率参数，典型值 1.7
- c、d：振幅参数，典型值 0.06 和 1.2

**视觉效果**
- 2D 平面上的复杂图案
- 丰富的纹理细节
- 自相似的分形结构

**应用场景**
- 生成艺术
- 纹理生成
- 视觉设计

### 4. Rossler 吸引子

**历史背景**
- 发现者：Otto Rössler（1976）
- 背景：研究化学反应中的混沌行为
- 意义：比 Lorenz 更简单的混沌系统

**数学方程**
```
dx/dt = -y - z
dy/dt = x + a*y
dz/dt = b + z*(x - c)
```

**参数说明**
- a：反馈系数，典型值 0.2
- b：偏置项，典型值 0.2
- c：阈值参数，典型值 5.7

**视觉效果**
- 螺旋状混沌系统
- 单翼结构
- 不断向外扩展的螺旋

**应用场景**
- 化学反应动力学
- 电子电路
- 生物节律

### 5. 混合模式（HYBRID）

**设计理念**
- 结合噪声场和吸引子的特点
- 保持随机性和结构性
- 创造独特的视觉效果

**混合算法**
```
velocity = noise_velocity * (1 - ratio) + attractor_velocity * ratio
```

其中 `ratio` 为混合比例，默认 0.3（30% 吸引子，70% 噪声场）

**视觉效果**
- 有机流动 + 结构性图案
- 更复杂的运动轨迹
- 独特的视觉风格

## 技术实现

### CPU 模式实现

**算法步骤**
1. 使用欧拉积分法计算吸引子速度
2. 根据运动模式选择对应的吸引子方程
3. 应用惯性因子平滑过渡
4. 更新粒子位置和速度
5. 边界检测和重置

**性能特点**
- 适用于小规模粒子（< 10,000）
- 精确的计算结果
- 灵活的参数调整

### GPU 模式实现

**着色器计算**
```glsl
vec3 lorenzAttractor(vec3 pos, float dt) {
  float sigma = uLorenzSigma;
  float rho = uLorenzRho;
  float beta = uLorenzBeta;
  
  float dx = sigma * (pos.y - pos.x);
  float dy = pos.x * (rho - pos.z) - pos.y;
  float dz = pos.x * pos.y - beta * pos.z;
  
  return vec3(dx, dy, dz) * dt;
}
```

**优势**
- 支持大规模粒子（100,000+）
- 并行计算所有粒子
- 硬件加速性能

**技术要点**
- 使用 GPUComputationRenderer
- 浮点纹理存储粒子数据
- Fragment Shader 并行计算
- 避免 CPU-GPU 数据传输

## 数值积分方法

### 欧拉积分（Euler Integration）

**公式**
```
x(t+dt) = x(t) + dx/dt * dt
```

**特点**
- 简单快速
- 一阶精度
- 适用于实时渲染

**局限性**
- 精度较低
- 可能有数值误差累积

### 龙格-库塔积分（RK4，可选升级）

**公式**
```
k1 = f(t, x)
k2 = f(t + dt/2, x + dt*k1/2)
k3 = f(t + dt/2, x + dt*k2/2)
k4 = f(t + dt, x + dt*k3)

x(t+dt) = x(t) + dt*(k1 + 2*k2 + 2*k3 + k4)/6
```

**特点**
- 四阶精度
- 更准确的计算结果
- 更稳定

**当前状态**
- 目前使用欧拉积分（性能优先）
- 可选升级到 RK4（精度优先）

## 参数调节

### Lorenz 参数

- **Sigma (σ)**：1-20
  - 小值：收敛性强，运动缓慢
  - 大值：发散性强，运动快速
  
- **Rho (ρ)**：10-50
  - 小值：混沌性弱
  - 大值：混沌性强
  
- **Beta (β)**：1-5
  - 控制蝴蝶形状的宽窄

### Thomas 参数

- **B**：0.01-0.5
  - 小值：阻尼小，振荡持续
  - 大值：阻尼大，快速收敛

### Clifford 参数

- **A、B**：0.5-3
  - 控制空间频率
  
- **C**：0.01-0.5
  - 控制振幅
  
- **D**：0.5-3
  - 控制相位偏移

### Rossler 参数

- **A**：0.01-0.5
  - 控制反馈强度
  
- **B**：0.01-0.5
  - 控制垂直方向偏置
  
- **C**：1-10
  - 控制混沌阈值

### 通用参数

- **时间缩放（Time Scale）**：0.0001-0.01
  - 控制计算步长
  - 小值：慢速运动
  - 大值：快速运动
  
- **粒子缩放（Particle Scale）**：0.001-0.1
  - 控制吸引子的大小
  - 小值：吸引子紧凑
  - 大值：吸引子分散

## 性能优化

### 计算优化

1. **GPU 并行计算**
   - 所有粒子同时计算
   - 避免循环开销
   - 利用 GPU 大量核心

2. **纹理缓存**
   - 使用浮点纹理存储
   - 减少 CPU-GPU 传输
   - 高速内存访问

3. **批处理更新**
   - 一次性更新所有粒子
   - 减少状态切换
   - 提高渲染效率

### 内存优化

1. **Float32Array**
   - 紧凑的数据存储
   - 高效的内存访问
   - 避免装箱拆箱

2. **纹理压缩**
   - 使用半精度浮点数
   - 减少内存占用
   - 保持足够精度

3. **对象池**
   - 重用临时对象
   - 减少垃圾回收
   - 降低内存分配
